# 로또 번호 추첨 사이트

## 개요

이 프로젝트는 사용자가 버튼을 클릭하여 로또 번호를 생성할 수 있는 간단한 웹 애플리케이션입니다.

## 기능

*   1부터 45까지의 숫자 중에서 6개의 고유한 랜덤 번호를 생성합니다.
*   생성된 번호를 사용자에게 시각적으로 표시합니다.
*   모던하고 매력적인 UI를 제공합니다.

## 현재 요청 계획

1.  **`index.html`**: 기본 HTML 구조를 설정합니다.
    *   사이트 제목 (`<h1>`)
    *   번호 생성 버튼 (`<button>`)
    *   생성된 번호를 표시할 영역 (`<div>`)
2.  **`style.css`**: 사이트의 디자인을 개선합니다.
    *   배경, 버튼, 번호 표시에 대한 스타일을 추가합니다.
3.  **`main.js`**: 로또 번호 생성 로직을 구현합니다.
    *   버튼 클릭 이벤트를 감지합니다.
    *   6개의 고유한 랜덤 번호를 생성하고 화면에 표시합니다.
